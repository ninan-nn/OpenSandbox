# Components

Core components (based on repo docs):

- **Server (Lifecycle API)**: FastAPI service for sandbox lifecycle (<https://github.com/alibaba/OpenSandbox/blob/main/server/README.md>)
- **execd (Execution API)**: in-sandbox daemon for code/command/files (<https://github.com/alibaba/OpenSandbox/blob/main/components/execd/README.md>)
- **Specs**: OpenAPI contracts (<https://github.com/alibaba/OpenSandbox/blob/main/specs/README.md>)
- **SDKs**: multi-language SDKs (`sdks/`)
- **Sandboxes**: runtime images and environments (`sandboxes/`)

Related components:

- **Kubernetes Controller** (<https://github.com/alibaba/OpenSandbox/blob/main/kubernetes/README.md>)
- **Ingress / Egress** (<https://github.com/alibaba/OpenSandbox/tree/main/components/ingress>, <https://github.com/alibaba/OpenSandbox/tree/main/components/egress>)
