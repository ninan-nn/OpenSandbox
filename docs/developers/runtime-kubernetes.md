# Runtime (Kubernetes)

Kubernetes Controller manages sandboxes using custom resources (see <https://github.com/alibaba/OpenSandbox/blob/main/kubernetes/README.md>).

## Key capabilities

- **BatchSandbox** for batch delivery
- **Pool** resources for pre-warmed capacity
- Optional task scheduling

See <https://github.com/alibaba/OpenSandbox/blob/main/kubernetes/README.md> for details and examples.
