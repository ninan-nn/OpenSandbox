# Server Configuration

This page summarizes server configuration based on repository docs.

## Config file

Default config file: `~/.sandbox.toml`.

```bash
opensandbox-server init-config ~/.sandbox.toml --example docker
```

Full config reference: <https://github.com/alibaba/OpenSandbox/blob/main/server/README.md>.

## Common fields (excerpt)

- `server.host`
- `server.port`
- `server.api_key`
