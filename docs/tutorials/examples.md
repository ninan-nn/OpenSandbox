# Examples

OpenSandbox provides examples for coding agents, browser automation, remote desktops, and training.

## Suggested steps

1. Pick a scenario
2. Read the example `README.md`
3. Set env vars and run `python main.py`

## Catalog

- `code-interpreter`, `aio-sandbox`, `host-volume-mount`
- `claude-code`, `gemini-cli`, `codex-cli`, `iflow-cli`, `langgraph`, `google-adk`
- `chrome`, `playwright`, `desktop`, `vscode`
- `rl-training`
