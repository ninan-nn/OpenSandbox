# 产品概览

OpenSandbox 是面向 AI 应用的通用沙箱平台，提供 **统一协议、可插拔运行时与多语言 SDK**，用于安全执行代码、命令与文件操作。它的设计目标是让开发者和平台团队以最小成本构建可控、可扩展的沙箱基础设施。

## 核心能力

- **统一协议**：生命周期 API + 执行 API（execd），规范化接口与模型
- **多语言 SDK**：Python、Kotlin/Java 与 Code Interpreter SDK
- **运行时可插拔**：Docker 生产可用，Kubernetes 可扩展
- **执行环境**：命令执行、文件系统、Jupyter 多语言内核
- **安全与治理**：API Key + execd 访问令牌，资源限制与 TTL
- **网络接入**：单机代理路由，统一端口映射与多端口访问

## 组件一览

OpenSandbox 由四层构成：

1. **SDK 层**：提供 Sandbox、Filesystem、Commands、CodeInterpreter 等高层抽象
2. **协议层**：OpenAPI 规范，定义生命周期与执行协议
3. **运行时层**：服务端实现生命周期 API，负责调度与资源管理
4. **实例层**：运行中的沙箱容器 + execd 代理

## 典型场景

- **AI 代码执行**：让模型生成的代码在隔离环境安全运行
- **工具执行平台**：提供命令、文件、代码执行 API 统一能力
- **远程开发与 GUI**：VS Code Web、桌面、浏览器自动化
- **评测与训练**：可控环境复现、并行执行与结果采集
