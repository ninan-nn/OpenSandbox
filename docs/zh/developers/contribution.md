# 贡献指南

详细流程见 <https://github.com/alibaba/OpenSandbox/blob/main/CONTRIBUTING.md>。这里仅列关键要点：

## 参与方式

- 提交 Issue / 需求建议
- 修复 Bug、提交功能
- 改进文档或测试

## 开发环境（摘要）

- Server：Python 3.10+ / uv / Docker
- execd：Go 1.24+ /（可选）Make / Docker
- SDK：Python / Java-Kotlin

## 提交流程

1. Fork 仓库并创建分支
2. 完成功能或修复
3. 运行测试与 lint
4. 提交 PR
