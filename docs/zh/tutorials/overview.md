# Tutorials 概览

本模块面向普通用户，提供从「快速上手」到「场景示例」的完整路径。

OpenSandbox 是面向 AI 应用的通用沙箱平台，提供 **统一协议、可插拔运行时与多语言 SDK**，用于安全执行代码、命令与文件操作。

## 推荐阅读顺序

1. `快速开始`：最短路径启动服务并执行命令/代码
2. `Server 配置`：了解服务端基础配置与常用字段
3. `Runtime 运行时`：了解 Docker/Kubernetes 与资源池、批量沙箱能力
4. `示例与场景`：选择与你业务贴近的案例
5. `常见问题 / 排障`：遇到问题先查看 FAQ 与排障指南
