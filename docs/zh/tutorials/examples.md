# 示例与场景

OpenSandbox 提供丰富示例覆盖编码代理、浏览器自动化、远程桌面、评测/训练等场景。

## 快速使用建议

1. 选择与你场景最接近的示例目录
2. 阅读该示例的 `README.md`
3. 配置环境变量并运行 `python main.py`

## 场景清单

### 基础能力

- `code-interpreter`：代码解释器 SDK 使用范式
- `aio-sandbox`：All-in-one 沙箱工作流
- `host-volume-mount`：挂载宿主目录

### Coding Agents

- `claude-code`
- `gemini-cli`
- `codex-cli`
- `iflow-cli`
- `langgraph`
- `google-adk`

### 浏览器与桌面

- `chrome`
- `playwright`
- `desktop`
- `vscode`

### 训练与评测

- `rl-training`

完整说明参考 <https://github.com/alibaba/OpenSandbox/blob/main/examples/README.md>。
